# Latest in Technical Writing Automation (TWA) - Learning Resources

A curated collection of cutting-edge tools, frameworks, and communities for **Agentic Documentation Pipelines** (as of April 2026).

---

## 1. Skill & Tool Libraries

### Antigravity Awesome Skills
- **Description:** A massive open-source library (1,200+ skills) used by agents like Claude Code
- **Specialty:** Includes specific skills for OpenAPI spec generation, Mermaid.js diagramming, and style guide enforcement
- **GitHub:** https://github.com/antigravity/awesome-skills
- **Status:** Primary standard for skill-based AI agent documentation

### MCP (Model Context Protocol) Servers
- **Description:** Current gold standard for documentation automation
- **Capabilities:** 
  - Direct integration with GitHub issues
  - Linear tickets browsing
  - Jira metadata pulling
  - Automatic release notes drafting
- **Repository:** Search on GitHub for "MCP servers documentation"
- **Status:** Recommended for production workflows

## 2. Agent Frameworks (The "Engine")

### CrewAI
- **Description:** Popular framework for building documentation agent teams
- **Use Case:** Compose "Crews" (teams of agents) for collaborative documentation
- **GitHub:** https://github.com/crewAIInc/crewAI
- **Best For:** API documentation automation, multi-role workflows
- **Stars:** 47.9k

### LangGraph
- **Description:** Framework for building agentic workflows
- **Use Case:** Complex state management and orchestration
- **GitHub:** https://github.com/langchain-ai/langgraph
- **Best For:** Advanced agentic pipelines
- **Stars:** 28.3k

### Dify
- **Description:** Low-code open-source platform for automation
- **Features:** Pre-built templates for API documentation automation and RAG (Retrieval-Augmented Generation)
- **GitHub:** https://github.com/langgenius/dify
- **Best For:** Rapid prototyping without extensive coding
- **Stars:** 136k

## 3. Prompt & Template Libraries

### Writerstable
- **Description:** Prompt engineering for technical writers
- **Specialty:** "Surgical Prompts" following formula: `Role + Goal + Context + Instruction + Constraints`
- **URL:** Search for "Writerstable" community
- **Best For:** Specialized, high-precision documentation prompts
prompts.chat (formerly Awesome-Prompts)
